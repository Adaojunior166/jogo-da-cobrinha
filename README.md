# Jogo da Cobrinha (Snake) em Python 🐍

Um clássico jogo da cobrinha desenvolvido em Python utilizando a biblioteca `pygame`. O projeto possui um visual modernizado, sistema de pontuação e aumento progressivo de dificuldade.

## 🎮 Funcionalidades

* **Visual Modernizado:** Fundo em *Dark Mode* com uma grade sutil para facilitar a noção espacial.
* **Sistema de Recorde:** O jogo salva a sua maior pontuação (High Score) enquanto a janela estiver aberta.
* **Dificuldade Progressiva:** A velocidade da cobra aumenta gradualmente conforme você coleta mais maçãs.
* **Controles Precisos:** Trava de segurança para evitar *Game Over* acidental ao pressionar duas teclas de direção muito rápido.

## 🛠️ Pré-requisitos

Para rodar este jogo, você precisará ter o [Python](https://www.python.org/downloads/) instalado na sua máquina, além da biblioteca Pygame.

## 🚀 Como executar o projeto

**1. Clone o repositório (ou baixe o arquivo `.py`):**
git clone https://github.com/SEU_USUARIO/NOME_DO_REPOSITORIO.git
cd NOME_DO_REPOSITORIO

**2. Instale a biblioteca Pygame:**
pip install pygame

**3. Execute o jogo:**
python nome_do_arquivo.py

*(Lembre-se de substituir `nome_do_arquivo.py` pelo nome real do seu arquivo Python).*

## ⌨️ Controles do Jogo

* **W, A, S, D** ou **Setinhas (⬆️, ⬅️, ⬇️, ➡️):** Movimentam a cobra.
* **R:** Reinicia a partida na tela de *Game Over*.
* **Q:** Sai do jogo.
